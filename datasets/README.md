# Dataset Information

## Original Dataset

Description:
The initial multimodal fake news dataset containing raw textual articles, metadata, and associated images collected from publicly available sources.

Kaggle Link: [<Original Dataset URL>](https://www.kaggle.com/datasets/ditioza/fakenewsproject)

---

## Preprocessed Dataset

Description:
A model-ready dataset generated through text cleaning, tokenization, image preprocessing, normalization, and feature preparation for machine learning and deep learning experiments.

Kaggle Link: [<Preprocessed Dataset URL>](https://www.kaggle.com/datasets/ditioza/sample)
