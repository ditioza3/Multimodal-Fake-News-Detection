# Multimodal Fake News Detection

## Overview

This project presents a multimodal fake news detection framework that leverages both textual and visual information to classify news articles as real or fake. The system combines Natural Language Processing (NLP) and Computer Vision techniques to improve misinformation detection performance compared to single-modality approaches.

## Problem Statement

Traditional fake news detection systems primarily rely on textual information. However, misleading news often includes manipulated images that influence credibility. This project explores the effectiveness of combining text and image features for fake news classification.

## Dataset

* Used Fakeedit dataset available on kaggle.
* Performed data cleaning, preprocessing, and feature extraction.
* Generated a sample dataset for model training and evaluation.

## Methodology

1. Text preprocessing and tokenization.
2. Feature extraction using transformer-based language models.
3. Image feature extraction using deep learning techniques.
4. Fusion of textual and visual representations.
5. Classification and evaluation using machine learning/deep learning models.

## Results

The multimodal approach demonstrated improved performance compared to unimodal baselines, highlighting the effectiveness of combining textual and visual information for misinformation detection.
